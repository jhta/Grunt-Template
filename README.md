Grunt-Template
==============

My Grunt template for web apps :D

Include:

* Browserify
* Bower
* Grunt Contribs:
  * Uglify
  * Jade
  * Stylus
  * minify JS, CSS and HTML

Install:

    npm install grunt-cli grunt bower -g
    npm install browserify grunt-browserify debowerify
    npm init 
    bower init
